# 哥特王朝

    version:  1
    created at 2016-01-22 15:18:53 

我玩的是2，乌鸦之夜。 画面其实不是很好，像素有点低。
然而游戏真是有气氛!

刚出城堡，路上就遇到狼。
再去有个小池塘，可以跳下去游泳， 甚至潜水下去。 非常有感觉。

游戏的地形可以用奇险来形容， 高山小道洞穴。
黑夜之时，一片黑暗，火把照路，胆战心惊。 
有条路上甚至有死尸挂在树上，令人害怕。 

游戏的内容也是十分丰富。
尤其是作弊。 可以变身为某人，脱掉其衣服，拿走所有装备。
可以穿墙，可以入地。

风景十分优美， 在主城旁边山顶某处，一个猎人坐在树干上看风景。

晚上客栈外面的火把，安慰着旅人。

我似乎对夜晚时，房子昏黄的灯光特别有亲切感。
