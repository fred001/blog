      <p>
      以PS项目为例：
项目的意义： 完成某个目标

功能需求：
1， 检测目标完成状况
2， 记录项目发展状况，为未来新项目做参考
3， 记录项目信息， 只要是未来有用途的
4， 项目可以随时获取当前状态


文件组织结构：
main.txt #记录项目情况
desc
user
target
status

log.txt #项目日志
target.txt #项目目标
