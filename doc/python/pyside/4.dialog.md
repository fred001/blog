# QDialog

对话框有至少两种用法。 
自带的众多对话框， 如文件选择， 颜色选择。
自己创建自定义的对话框.


#### QColorDialog 颜色选择对话框
  dialog=QColorDialog(parent=widget) //创建 对话框
  print dialog.getColor() //打开对话框并获得结果

#### QFileDialog  文件选择对话框
  dialog=QFileDialog(parent=self)
  if dialog.exec_():
    print dialog.selectedFiles()
